Information Security

In Information Security my job the was Information Security Analyst. My job description was to be responsible for planning and implementing security measures to protect computer systems, networks, and data. To be responsible for staying up-to-date on the latest intelligence and hacking methodologies, in order to anticipate security breaches, as well as researching new technologies that will effectively protect a network and prevent data loss and service interruptions. My first assignment was to look up and research password management systems that we could use. Our requirements were that it had to be free to use, be secure, and easy to use. I found one that we could use and told the System Administrator about it and we decided to use it. We got that downloaded and the next thing I had to do was make a slide describing how to use it. I made the slide saying what it was, why were using, and how to use it. I presented this to the rest of the company so they knew how to use it.

Business Intelligence

I was the Data Analyst for Business Intelligence and my job description was to interpret data, analyze results using statistical techniques, developing and implementing data analyses, data collection systems and other strategies that optimize statistical efficiency and quality. My goal was to turn data into information, information into insight and insight into business decisions. My first project was to get all of the data from the website and making it into slides so we can present it to the Manager of the business. Once we completed that our second project was to learn how to use Qlik sense and Data Studios. Once we learned how to use them we made slides to choose which one is better to use. Once we made the slides we presented it to the Manager to choose which one we should use and we decided that we would use Data Studio. After we chose Data Studio we made data from the website for all of the teams in the business and chose which data each team could see and use.
